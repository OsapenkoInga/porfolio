Сайт #4

Сайт содержит боковую панели слева,которая содержит header с логотипом, вертикильное меню(nav) и подвал(footer). При наведении мыши, пункт меню меняет фон.При наведении на about us всплывает еще одно подменю. Подвал сайта состоит из списка социконок и копирайтов.При нажатии на пункт меню появляется основное содержимое(article),а в нем секция, имя класса которой совпадает с выбраным пунктом.Все это сделано с помощью JS, принцип работы таков: при первом клике на пункт меню свойство visibility, блока article, становится visible,при повторном клике visibility:hidden и через 400 милисекунд опять visible.Секция services содержит таблицы, строки которых чередуются разными backgraund-ами.Секция gallery содержит галерею изображений, которую я сделала с помощью jQuery.
